## For building our application we will be using Maven


 1. git clone https://github.com/devopshydclub/vprofile-project.git
 2. cd vprofile-project/
 3. cd src
 4. git branch -a
 5. git check aws-Refactor
 6. cd main/resources/
 7. vi application.properties               
 9. cd ../../ ..
 10. mvn install
